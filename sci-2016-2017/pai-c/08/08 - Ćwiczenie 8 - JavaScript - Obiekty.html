Obiekty

Obiekty są to konstrukcje programistyczne posiadające tzw. właściwości, którymi mogą być zmienne JavaScript'u lub inne obiekty. Z obiektami powiązane są funkcje wykonujące operacje na właściwościach obiektu, które nazywamy metodami.

W JS w przeciwieństwie do innych języków nie mamy mechanizmu konstruktora, ale samą klasę możemy stworzyć za pomocą zwykłej funkcji:

function AaBbCc(_x,_y) { this.x = _x; this.y = _y; this.print = function() { console.log(this.x + this.y) } }

Do właściwości danego obiektu możemy się "dostać", używając następującej notacji:

nazwa_obiektu.nazwa_właściwości

Np. jeśli mamy obiekt "komputer", może on mieć właściwości: procesor, zegar, pamiec, cena. Aby więc "wypełnić" obiekt odpowiednimi wartościami, będziemy musieli wykonać następującą sekwencję instrukcji:

komputer.procesor = "i7"
komputer.zegar = 2766
komputer.pamiec = 4096
komputer.cena = "2500 zł"

Identyczny efekt uzyskamy również przy zastosowaniu innej notacji:

nazwa_obiektu[nazwa_właściwości], np. komputer[procesor]

Pozostaje pytanie, jak taki obiekt stworzyć? Najpierw musimy go zdefiniować, tworząc funkcję zwaną konstruktorem, a następnie utworzyć za pomocą operatora: new

var mojObiekt = new AaBbCc(10, 20); mojObiekt.drukuj(); wypisze 1020

Zadanie 8A

Napisz funkcję konstruktora obiektu o nazwie komputer. Utwórz obiekt za pomocą operatora new, a następnie wyświetl jego właściwości na ekranie.

Częściami składowymi obiektu nie muszą być tylko "zwykłe" zmienne. Mogą nimi również być inne obiekty. Jeśli chcielibyśmy do obiektu komputer dodać informację o jego właścicielu, warto stworzyć nowy obiekt - właściciel - i potraktować go jako właściwość komputera.
