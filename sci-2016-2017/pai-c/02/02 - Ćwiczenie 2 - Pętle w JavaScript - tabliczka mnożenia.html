Pętle w JavaScript - tabliczka mnożenia

Wykorzystując przykład podany na poprzednim slajdzie napisz funkcje wyświetlającą na ekranie tabliczkę mnożenia.

Przykład z poprzedniego slajdu: 

for (i = 0; i < 5; i++) {
   document.writeln(„jestem w petli zewnętrznej” + i);
   for (j = 0; j<5; j++) {
     document.writeln(„jestem w pętli wewnętrznej „+j);
   }
}
