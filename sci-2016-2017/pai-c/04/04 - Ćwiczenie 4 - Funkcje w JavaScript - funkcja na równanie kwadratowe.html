Funkcje w JavaScript - funkcja na równanie kwadratowe

Napisz funkcję obliczającą pierwiastki równania kwadratowego w dzidzinie liczb rzeczywistych (wykorzystaj algorytm z Ćwiczenia 1). Funkcja powinna przyjmować różne wartości (nie tylko bezpośrednio z formularza), sprawdzać co do niej wprowadzono i zwracać wynik w postaci tablicy, wartości logicznej true/false lub liczby w zależności od przypadku. Rozważ wszystkie możliwe przypadki.

Wywołanie funkcji i przechwycenie wyniku może nastąpić w dowolny sposób (dane z formularza, dane z innej funkcji, dane losowo wybrane).

Możesz użyć pliku HTML z poprzedniego zadania, utwórz jednak nowy plik JS i dołącz do dokumentu.

Przydatne rzeczy:

Ax2 + Bx + C = 0

dla A nie będącego zerem, sprawdź to!!
Przydatne może być również: 
isNaN
definicja funkcji: 
function kwadratowe(a,b,c){
	//instrukcje wewnątrz funkcji
	return(zwracaneWartosci);	
}
oraz wywołanie funkcji: 
kwadratowe(1,2,3);
pozostaje tylko przechwycenie wyniku i jego odpowiednie wyświetlenie.
