Instrukcje warunkowe w JavaScript - równanie kwadratowe

Napisz skrypt wyświetlający rozwiązania równania kwadratowego: 

Ax2 + Bx + C

Celem ćwiczenia jest praktyczne zapoznanie się z instrukcjami warunkowymi. Do tego celu wykorzystamy algorytm obliczania równania kwadratowego Ax2 + Bx + C. Równanie ma rozwiązanie, jeśli parametr Δ (delta) równy B2–4(A*C) jest większy od 0 lub mu równy. Jeśli Δ równa się 0, mamy jedno rozwiązanie równe –B/(2*A), jeśli Δ jest większa od 0, mamy dwa rozwiązania: x1 = (–B+√Δ)/(2∗A) i x2 = (–B–√Δ)/(2∗A).
