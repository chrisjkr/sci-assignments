Obliczenia i funkcje w JavaScript - podzielność liczb

Napisz skrypt JS, który sprawdza czy wprowadzone (lub zapisane w skrypcie) przez użytkownika liczby są parzyste czy nieparzyste (inaczej czy są podzielne lub nie przez 2).

Użyj funkcji jak najbardziej uniwersalnej, czyli takiej którą możesz wykorzystać do sprawdzenia podzielności przez 3, 5, 7 lub dowolną inną liczbę.

Skrypt osadź w zewnętrznym pliku js i dołącz do pliku html.

Funkcja powinna zwracać true/false w zależności od wyniku.

Wskazówka

Wykorzystaj instrukcję warunkową i operator modulo.

Przykłady funkcji

function mojaFunkcja1(p1, p2) {
return p1 * p2; // funkcja zwraca wynika mnożenia p1 i p2
}

function mojaFunkcja2(p1, p2) {
     var wynik = p1 * p2;    //mnożenie p1 i p2
     return wynik; // funkcja zwraca wynik
}
